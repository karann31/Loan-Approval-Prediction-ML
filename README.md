# 🏦 Loan Approval Prediction using Machine Learning & Flask

An end-to-end Machine Learning project that predicts whether a loan application will be **Approved** or **Rejected** based on applicant details. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, hyperparameter tuning, and deployment as a Flask web application.

---

## 📌 Project Overview

Financial institutions receive thousands of loan applications every day. Evaluating each application manually is time-consuming and prone to inconsistencies. This project leverages Machine Learning to automate the loan approval process by predicting loan eligibility based on applicant information.

The trained model is deployed using **Flask**, providing an interactive web interface where users can enter applicant details and instantly receive a loan approval prediction.

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Loan-Approval-Prediction-ML.git
```

Move into the project directory

```bash
cd Loan-Approval-Prediction-ML
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Flask application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
